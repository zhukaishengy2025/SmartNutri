# SmartNutri 宠粮匹配度报告 — 简化版方案设计

> 版本：v2.0-lite  
> 日期：2026年3月  
> 定位：在保留科学依据的前提下，大幅精简运算逻辑，降低计算量，适合 MVP 快速上线

---

## 与 v1.0 的关键差异

| 对比项 | v1.0（完整版） | v2.0-lite（简化版） | 精简效果 |
|-------|-------------|-------------------|---------|
| 评分函数 | 6 阈值梯形函数（7 段分支插值） | 3 区间线性函数（3 段） | 运算分支减半 |
| 营养素阈值参数 | ~180 个（6 阈值 × 5 指标 × 6 生命阶段） | ~36 个（2 阈值 × 3 指标 × 6 生命阶段） | **减少 80%** |
| 维度内子因子 | 3-5 个/维度，各带独立权重 | 1-2 个/维度，直接查表或布尔判定 | 消除子因子加权运算 |
| 动态加权 | 10 条件 × 6 维度 = 60 个 ΔW，含归一化 + 下限约束 + 再归一化 | 乘数法：每个条件只触发 1-2 个维度的倍率放大，一次归一化 | **参数从 60 → 12** |
| 风险预警规则 | 20 条 | 10 条 | 减半 |
| 总可配置参数 | ~200+ | ~60 | **减少 70%** |

> 科学依据（AAFCO 2024 / NRC 2006 / 兽医营养学文献）完全保留，仅精简计算路径。

## 一、整体流程

```
宠物档案 + 宠粮成分
        │
        ▼
  ┌─ 预处理 ──────────────────┐
  │  DMB 转换 · NFE 反算 · ME │
  └────────────┬──────────────┘
               ▼
  ┌─ 六维评分（简化） ────────┐
  │  D1 营养  D2 安全  D3 适口│
  │  D4 皮毛  D5 关节  D6 消化│
  └────────────┬──────────────┘
               ▼
  ┌─ 乘数加权 → 总分 ────────┐
  │  条件触发 → 倍率 → 归一化 │
  └────────────┬──────────────┘
               ▼

```

预处理阶段（DMB 转换、NFE 反算、ME 估算）与 v1.0 完全相同，此处不再重复。

## 二、简化评分函数

### v1.0 的问题

v1.0 使用 6 阈值梯形函数（CL/AL/OL/OH/AH/CH），每个营养素需要 7 段分支判断。对于 5 个营养素 × 6 个生命阶段组合 = 30 组阈值，每组 6 个参数，共计约 180 个阈值参数。

### v2.0 方案：3 区间线性函数

每个营养素只需 **2 个参数**：推荐下限 (MIN) 和推荐上限 (MAX)。

```
Score
100 ┤     ┌──────────────────┐
    │    ╱                    ╲
    │   ╱                      ╲
    │  ╱                        ╲
  0 ┤─╱                          ╲─
    └─┬──────┬──────────┬──────┬──→
      0     MIN        MAX    MAX×2
```

**公式：**

$$
S(x) = \begin{cases}
\dfrac{x}{\text{MIN}} \times 100 & x \leq \text{MIN} \\[6pt]
100 & \text{MIN} < x \leq \text{MAX} \\[6pt]
\max\!\left(0,\; 100 - \dfrac{x - \text{MAX}}{\text{MAX}} \times 100\right) & x > \text{MAX}
\end{cases}
$$

- 低于 MIN：线性衰减至 0
- MIN~MAX 之间：满分 100
- 高于 MAX：线性衰减（以 MAX 等宽的距离衰减至 0）

> 只有 3 个分支，无需查找 6 个阈值。

## 三、六维评分（简化版）

### D1 — 营养均衡

#### 阈值表（仅需 MIN 和 MAX，单位 DMB%）

**蛋白质：**

| 物种-阶段 | MIN | MAX | 参考 |
|---------|-----|-----|------|
| 犬-幼年 | 22.5 | 40 | AAFCO min 22.5% |
| 犬-成年 | 18 | 35 | AAFCO min 18% |
| 犬-老年 | 25 | 40 | 老年犬需更高优质蛋白 |
| 猫-幼年 | 30 | 55 | AAFCO min 30% |
| 猫-成年 | 26 | 50 | AAFCO min 26% |
| 猫-老年 | 30 | 50 | 老年猫蛋白需求上调 |

**脂肪：**

| 物种-阶段 | MIN | MAX | 参考 |
|---------|-----|-----|------|
| 犬-幼年 | 8.5 | 22 | AAFCO min 8.5% |
| 犬-成年 | 5.5 | 18 | AAFCO min 5.5% |
| 犬-老年 | 5.5 | 15 | 老年犬脂肪上限收窄 |
| 猫-幼年 | 9 | 28 | AAFCO min 9% |
| 猫-成年 | 9 | 25 | AAFCO min 9% |
| 猫-老年 | 9 | 22 | 老年猫脂肪上限收窄 |

**碳水化合物 (NFE)：** 碳水无下限惩罚，仅有上限。

| 物种 | MIN | MAX | 参考 |
|-----|-----|-----|------|
| 犬 | 0 | 50 | 超过 50% 过高 |
| 猫 | 0 | 30 | 猫为专性肉食动物 |

#### 计算

$$
D_1 = \text{avg}\!\left(S_{\text{蛋白}},\; S_{\text{脂肪}},\; S_{\text{碳水}}\right)
$$

3 个子因子直接取算术平均，无需内部加权。

> **与 v1.0 对比：** 去掉了钙磷比和牛磺酸两个子因子（移入安全性维度作为风险检查项）。子因子从 5 个降至 3 个，取消子因子内部权重。

---

### D2 — 安全性

安全性仍采用 **扣分制**（基础 100，逐项扣除），但大幅精简扣分项。

#### 扣分规则（简化版）

| 类别 | 条件 | 扣分 | 说明 |
|-----|------|------|------|
| **致敏原** | 配料表含宠物已知过敏原 | **-50** / 项 | v1.0 区分直接/关联/同族 3 级。简化为：命中即扣 50。判定逻辑：配料名称包含过敏原关键词即判定命中（鸡肉过敏 → 含"鸡"字的任何配料命中） |
| **疾病禁忌** | 有 CKD 且磷 >1.0% DMB | **-30** | |
| | 有胰腺炎且脂肪 >15% DMB（犬）/ >25%（猫） | **-30** | |
| | 有糖尿病且碳水 >40% DMB（犬）/ >20%（猫） | **-25** | |
| **危险添加剂** | 含 BHA / BHT / 乙氧基喹 / 丙二醇(猫) | **-15** / 项 | 合并为一组，统一扣分值 |
| **猫粮缺牛磺酸** | 猫粮配料表未检测到牛磺酸 | **-30** | 从 D1 移入 |
| **生命阶段不匹配** | 幼年宠物 + 成犬粮 | **-20** | |
| | 其他不匹配 | **-5** | |

$$
D_2 = \max(0,\; 100 - \sum \text{扣分})
$$

> **与 v1.0 对比：** 致敏原匹配从 3 级（直接/关联/同族，扣 40/20/10）简化为 1 级（命中即扣 50）。消除了致敏原关联矩阵的维护成本。疾病禁忌从 8 条保留最高频 3 条。去掉了蛋白源质量检查（移至适口性维度间接体现）。

---

### D3 — 适口性

查表法，无需加分累计。

| 第一配料类型 | 基础分 | 前 5 配料含 ≥2 种命名动物源 | 最终 D3 |
|-----------|-------|-------------------------|--------|
| 命名鲜肉（如"去骨鸡肉"） | 80 | +20 → 100 | 80 或 100 |
| 命名肉粉（如"鸡肉粉"） | 60 | +20 → 80 | 60 或 80 |
| 谷物 / 植物蛋白 | 30 | +20 → 50 | 30 或 50 |

$$
D_3 = \text{base} + (\text{前 5 配料含 ≥2 种命名动物源} \;?\; 20 : 0)
$$

一次查表 + 一次布尔判断，无累加运算。

> **与 v1.0 对比：** 从 8 个加分项（涉及逐项遍历前 5 配料、脂肪区间判断、风味增强剂判断等）简化为 2 个判断。

---

### D4 — 皮肤美毛

查表法。

| 配料表中是否含 Omega-3 来源 | 得分 |
|--------------------------|------|
| 含鱼油（三文鱼油、鳀鱼油等） | 60 |
| 含亚麻籽 / 亚麻籽油（但无鱼油） | 35 |
| 无可识别 Omega-3 来源 | 10 |

| 功能性成分加分 | 条件 | 加分 |
|-------------|------|------|
| 含锌补充剂 | 配料表含"锌" | +15 |
| 含维生素 E | 配料表含"生育酚"或"维生素 E" | +10 |
| 含生物素 | 配料表含"生物素"或"biotin" | +10 |

$$
D_4 = \min(100,\; S_{\text{Omega-3 查表}} + \text{功能性加分})
$$

> **与 v1.0 对比：** 去掉了 Omega-6:Omega-3 比值计算（需要标签标注 Omega 数据，实际常缺失）。从 3 个加权子因子简化为查表 + 简单加分。

---

### D5 — 骨骼关节

查表法。

| 配料表中的关节支持成分 | 得分 |
|--------------------|------|
| 含氨基葡萄糖 **且** 含硫酸软骨素 | 60 |
| 仅含其中一种 | 40 |
| 含绿唇贻贝 | 50 |
| 无任何关节支持成分 | 0 |

| Omega-3 抗炎加分 | 条件 | 加分 |
|----------------|------|------|
| 含鱼油来源 | 配料表含鱼油 | +30 |
| 含亚麻籽 | 配料表含亚麻籽（ALA，非直接抗炎） | +10 |

$$
D_5 = \min(100,\; S_{\text{关节查表}} + S_{\text{Omega-3 加分}})
$$

> **与 v1.0 对比：** 去掉了氨基葡萄糖含量的梯形函数评分（需 mg/kg 数据，标签常缺失）。去掉了 MSM、维 C 等辅助成分评分。去掉了幼年犬的钙磷比切换逻辑。从 3 个加权子因子 + 年龄分支简化为查表 + 加分。

---

### D6 — 消化健康

查表法 + 一次线性计算。

**粗纤维评分（简化线性函数）：**

| 物种 | MIN | MAX |
|-----|-----|-----|
| 犬 | 2.0% DMB | 5.0% DMB |
| 猫 | 1.5% DMB | 4.0% DMB |

$$
S_{\text{纤维}} = S(x, \text{MIN}, \text{MAX})
$$

**肠道健康成分加分：**

| 条件 | 加分 |
|------|------|
| 含益生元（甜菜粕 / 菊苣根 / FOS / 菊粉） | +25 |
| 含益生菌（任何标注的菌株） | +25 |

$$
D_6 = \frac{S_{\text{纤维}} + \min(100,\; 50 + \text{肠道加分})}{2}
$$

其中第二项的基础分 50 代表"无肠道健康成分时的基础消化率"。

> **与 v1.0 对比：** 去掉了蛋白消化率代理评分（依赖配料排序推断，逻辑复杂且主观）。子因子从 4 个降至 2 个。

## 四、动态加权（简化版）

### v1.0 的问题

v1.0 使用 10 条件 × 6 维度的 ΔW 矩阵（60 个参数），多条件叠加后需归一化 → 下限约束 → 再归一化，共 3 步标准化运算。

### v2.0 方案：乘数法

**基础权重（固定）：**

| 维度 | D1 营养 | D2 安全 | D3 适口 | D4 皮毛 | D5 关节 | D6 消化 |
|-----|--------|--------|--------|--------|--------|--------|
| 基础权重 | 25 | 25 | 10 | 13 | 13 | 14 |

**条件乘数：** 存在对应条件时，将目标维度权重乘以指定倍率。

| 条件 | 目标维度 | 乘数 | 说明 |
|-----|---------|------|------|
| 有食物过敏 | D2 安全性 | ×2.0 | 致敏原匹配变为最优先 |
| 有关节问题 | D5 关节 | ×2.5 | 关节支持变为核心诉求 |
| 有皮肤/毛发问题 | D4 皮毛 | ×2.0 | 皮毛营养优先 |
| 有消化问题 (IBD 等) | D6 消化 | ×2.0 | 消化友好度优先 |
| 有 CKD / 胰腺炎 / 糖尿病 | D2 安全性 | ×2.0 | 饮食禁忌变为刚性约束 |
| 老年期 | D5 关节 | ×1.5 | 老年关节退化 |

**运算步骤（仅 2 步）：**

```
步骤 1：对每个适用条件，将基础权重中的目标维度 × 乘数
步骤 2：将所有权重归一化至总和 = 100
```

$$
W_i^{final} = \frac{W_i^{raw}}{\sum_{k=1}^{6} W_k^{raw}} \times 100
$$

> 不需要 ΔW 矩阵、不需要下限约束、不需要多轮归一化。

### 示例：9 岁金毛，鸡肉过敏 + 关节炎

条件：老年期 + 有食物过敏 + 有关节问题

| 维度 | 基础权重 | 应用乘数 | 计算后 | 归一化 (%) |
|-----|---------|---------|-------|-----------|
| D1 营养 | 25 | — | 25 | **17.2** |
| D2 安全 | 25 | ×2.0（过敏） | 50 | **34.5** |
| D3 适口 | 10 | — | 10 | **6.9** |
| D4 皮毛 | 13 | — | 13 | **9.0** |
| D5 关节 | 13 | ×2.5（关节）×1.5（老年）= ×3.75 | 48.75 | **33.6** |
| D6 消化 | 14 | — | 14 | **9.7** |
| **合计** | 100 | | 145.15 ← 仅用于归一化 | **≈100** |

> 安全性（34.5%）+ 关节（33.6%）= 68.1%，与 v1.0 的 67.3% 接近，但运算量从矩阵运算降至 3 次乘法 + 1 次归一化。同一条件触发多个维度的乘数时，乘数直接相乘（如关节 ×2.5 × ×1.5 = ×3.75）。

## 五、总分计算

与 v1.0 相同：

$$
\text{MatchScore} = \sum_{i=1}^{6} W_i^{final} \times D_i \div 100
$$

### 安全性一票否决（保留）

$$
\text{若 } D_2 \leq 30\text{，则 MatchScore} = \min(\text{MatchScore},\; 39)
$$

## 七、验证示例

使用与 v1.0 报告样例相同的输入，验证简化版算法的输出合理性。

### 输入

**宠物：** 金毛寻回犬，9 岁（老年期），20 kg，公，已绝育，鸡肉过敏，关节炎  
**宠粮：** 皇家大型犬成犬粮

| 指标 | DMB 值 |
|-----|--------|
| 蛋白质 | 27.8% |
| 脂肪 | 15.6% |
| 碳水 NFE | 47.2% |
| 粗纤维 | 1.67% |
| ME | 3,553 kcal/kg |

配料表前 5 项：小麦、鸭肉粉、鸡肉粉、豆粕、小麦粉  
其他关键配料：鸡油、牛油、大米、甜菜粕、大豆油  
添加剂含：硫酸锌、DL-α-生育酚（维E）、生物素  
无鱼油、无亚麻籽、无氨基葡萄糖/软骨素、无益生菌

### D1 — 营养均衡

阈值组：犬-老年

| 子因子 | 值 | MIN | MAX | 计算 | 得分 |
|-------|-----|-----|-----|------|------|
| 蛋白质 | 27.8% | 25 | 40 | 25 < 27.8 < 40 → 满分 | 100 |
| 脂肪 | 15.6% | 5.5 | 15 | 15.6 > 15（MAX），超出 0.6 → 100-0.6/15×100 = 96 | 96 |
| 碳水 | 47.2% | 0 | 50 | 0 < 47.2 < 50 → 满分 | 100 |

$$D_1 = (100 + 96 + 100) \div 3 = \boxed{99}$$

### D2 — 安全性

| 扣分项 | 匹配？ | 扣分 |
|-------|-------|------|
| 致敏原：鸡肉过敏 × 鸡肉粉 | ✅ 命中（含"鸡"） | -50 |
| 致敏原：鸡肉过敏 × 鸡油 | ✅ 命中（含"鸡"） | -50 |
| 疾病禁忌 | 关节炎无饮食禁忌 | 0 |
| 危险添加剂 | 未检测到 | 0 |
| 生命阶段 | 老年犬 + 成犬粮 → 其他不匹配 | -5 |

$$D_2 = \max(0,\; 100 - 50 - 50 - 5) = \max(0,\; -5) = \boxed{0}$$

> D2 = 0 ≤ 30，触发一票否决机制。

### D3 — 适口性

- 第一配料 = 小麦（谷物） → 基础分 30
- 前 5 配料：鸭肉粉(#2) + 鸡肉粉(#3) = 2 种命名动物源 ≥ 2 → +20

$$D_3 = 30 + 20 = \boxed{50}$$

### D4 — 皮肤美毛

- Omega-3 来源：无鱼油、无亚麻籽 → 查表 = 10
- 功能性加分：硫酸锌(+15) + 维E(+10) + 生物素(+10) = +35

$$D_4 = \min(100,\; 10 + 35) = \boxed{45}$$

### D5 — 骨骼关节

- 关节支持成分：无氨基葡萄糖、无软骨素、无绿唇贻贝 → 查表 = 0
- Omega-3 加分：无鱼油、无亚麻籽 → +0

$$D_5 = \min(100,\; 0 + 0) = \boxed{0}$$

### D6 — 消化健康

- 粗纤维 = 1.67% DMB，犬 MIN = 2.0%，1.67 < 2.0 → S = 1.67/2.0 × 100 = 84
- 肠道加分：含甜菜粕（益生元 +25）、无益生菌(+0) → 加分 = 25
- 第二项 = min(100, 50 + 25) = 75

$$D_6 = (84 + 75) \div 2 = \boxed{80}$$

### 动态加权

条件：有食物过敏 + 有关节问题 + 老年期

| 维度 | 基础 | 乘数 | 计算后 | 归一化 (%) |
|-----|------|------|-------|-----------|
| D1 | 25 | — | 25.0 | 17.2 |
| D2 | 25 | ×2.0（过敏） | 50.0 | 34.5 |
| D3 | 10 | — | 10.0 | 6.9 |
| D4 | 13 | — | 13.0 | 9.0 |

## 八、算法伪代码

```python
def calculate_match_score(pet, food):
    # ── 预处理 ──
    dmb = convert_to_dmb(food)
    nfe = calc_nfe(dmb)
    stage = get_life_stage(pet.species, pet.breed, pet.age_months)

    # ── 阈值查表 ──
    thresholds = THRESHOLDS[pet.species][stage]

    # ── D1 营养均衡 ──
    s_protein = score_linear(dmb.protein, thresholds.protein_min, thresholds.protein_max)
    s_fat     = score_linear(dmb.fat,     thresholds.fat_min,     thresholds.fat_max)
    s_carb    = score_linear(nfe,         0,                      thresholds.carb_max)
    d1 = (s_protein + s_fat + s_carb) / 3

    # ── D2 安全性 ──
    d2 = 100
    for allergen in pet.allergies:
        hits = count_keyword_hits(food.ingredients, allergen)
        d2 -= hits * 50
    d2 -= check_disease_contraindications(pet.conditions, dmb)
    d2 -= check_dangerous_additives(food.ingredients)
    d2 -= check_life_stage_mismatch(pet, food)
    d2 = max(0, d2)

    # ── D3 适口性 ──
    d3 = palatability_lookup(food.first_ingredient_type)
    if count_named_animal_sources_in_top5(food.ingredients) >= 2:
        d3 += 20

    # ── D4 皮毛 ──
    d4 = omega3_source_lookup(food.ingredients)
    d4 += functional_skin_bonus(food.ingredients)
    d4 = min(100, d4)

    # ── D5 关节 ──
    d5 = joint_supplement_lookup(food.ingredients)
    d5 += omega3_anti_inflam_bonus(food.ingredients)
    d5 = min(100, d5)

    # ── D6 消化 ──
    s_fiber = score_linear(dmb.fiber, thresholds.fiber_min, thresholds.fiber_max)
    gut_bonus = prebiotic_bonus(food.ingredients) + probiotic_bonus(food.ingredients)
    d6 = (s_fiber + min(100, 50 + gut_bonus)) / 2

    # ── 动态加权（乘数法） ──
    weights = BASE_WEIGHTS.copy()
    for condition in get_active_conditions(pet):
        for target, multiplier in CONDITION_MULTIPLIERS[condition]:
            weights[target] *= multiplier
    weights = normalize_to_100(weights)

    # ── 总分 ──
    scores = [d1, d2, d3, d4, d5, d6]
    total = sum(w * s for w, s in zip(weights, scores)) / 100

    # ── 一票否决 ──
    if d2 <= 30:
        total = min(total, 39)

    return total, scores, weights


def score_linear(value, min_val, max_val):
    """3 区间线性评分"""
    if value <= min_val:
        return (value / min_val * 100) if min_val > 0 else 100
    elif value <= max_val:
        return 100
    else:
        return max(0, 100 - (value - max_val) / max_val * 100)
```
